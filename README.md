# Personal Fullstack Developer Portfolio

This is a personal portfolio website built for Unit 1 of Advanced Web Development.

## Project Description

This website showcases my blogs, projects, and fullstack development learning journey. It is served by a Node.js and Express backend and uses static HTML, CSS, and JavaScript on the frontend.

## Tech Stack

- HTML
- CSS
- JavaScript
- Node.js
- Express
- PicoCSS

## Features

- Home page
- About page
- Blogs page
- Projects page
- Contact page
- Static files served by Express
- Shared blog and project data structure for future database integration

## How to Run

```bash
npm install
npm run dev
